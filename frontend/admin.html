Notification Config

Notifications are configured via environment variables on the VPS. Set these in /var/www/cfb-rankings/.env

SLACK_WEBHOOK_URL Slack incoming webhook URL — posts update summary to your channel
NOTIFY_EMAIL Email address to receive update summaries (requires mail on server)
CFBD_API_KEY CollegeFootballData.com API key (required for all imports)
ADMIN_SECRET Admin key protecting this dashboard and the import API endpoints
